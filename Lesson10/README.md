### 屏幕大小的划分

Bootstrap4根据客户端显示屏幕的大小，分为以下几个分类：

* 超小（Extra small）
  - <576px
  - .col-*
* 小（Small）
  - ≥576px
  - .col-sm-*
* 中（Medium）
  - ≥768px
  - .col-md-*
* 大（Large）
  - ≥992px
  - .col-lg-*
* 超大（Extra large）
  - ≥1200px
  - .col-xl-*
